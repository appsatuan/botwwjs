## Prerequisites
- Node.js
- npm

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/appsatuan/botwwjs.git
    ```
2. Navigate to the project directory:
    ```bash
    cd botwwjs
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

## Usage

1. Start the application:
    ```bash
    npm start
    ```
    or 
    ```
    ./manage-bot.sh
    ```
2. Access the bot at your desired location.
